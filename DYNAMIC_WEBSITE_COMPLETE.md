## 🚨 TROUBLESHOOTING

### Issue 1: Content नहीं दिख रहा
```
Solution:
1. Backend running है check करें
2. Browser console में errors देखें
3. API_KEY match हो रहा है check करें
```

### Issue 2: Old content दिख रहा है
```
Solution:
1. Browser cache clear करें (Ctrl+Shift+Delete)
2. Hard refresh करें (Ctrl+F5)
3. Incognito mode में try करें
```

### Issue 3: Backend error आ रहा है
```
Solution:
1. backend.log file check करें
2. Database connection verify करें
3. API_KEY backend .env में check करें
```
